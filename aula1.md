# Aula 1- Roteiro de Investigação( Tipos de Vírus e como detectalos )


## Roteiro de investigação
1- Quais os sintomas você observou primeiro?
2- Qual ferramenta usou para investigar?
3- O que você encontrou?
4- Qual tipo de Malware foi indentificado? Por quê?


## Vírus que eu e meu grupo trabalhou Keylogger

1- Sintomas observados: O computador não apresentava lentindão muito menos super uso de seu Hardware fisico como CPU,GPU,Memoria Ram e armazenamento 

2- A ferramenta usada: Gerenciador de Tarefas se fosse no linux Htop ou similar

3- O que foi encontrado: Um arquivo de LOG com todas as credenciais e escrita feita pelo usuario da maquina estava salvo lá

4- Keylogger: arquivos de logs crencendo
