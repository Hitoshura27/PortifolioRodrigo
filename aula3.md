# Manuntenção de Softwares no Linux Mint (Valdemir)

1- Como instalar aplicativos pelo Linux?
2- Como atualizar eles?
3- Como desistala-los?



## Linux Mint 21.3 

1- No linux existem 3 formas de instalar aplicativos sendo elas pela interface gráfica usando o gerenciador de aplicativos, terminal ou linha de comando usando o comando sudo apt install nome do aplicativo ou de forma de arquvos .deb que você baixa pela internet

2- A forma de atualiza-los pode ser através da linha de comando com o comando sudo apt update nome do programa ou você pode usar o sudo apt upgrade que te permite atualizar e corrigir pendencias também ou de maneira mais simples pelo gerenciador de aplicativos que geralmente já atualiza automaticamente 


3- Pela linha de comando você pode usar o sudo apt purge e pela interface grafica você pesquisa o aplicativo e depois clicka na opção de desistalar
